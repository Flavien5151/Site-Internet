Application de gestion

J’ai réalisé ce projet d'application dans le cadre d'un projet scolaire pour la gestion d'une entreprise.

L’objectif était de pouvoir gérer le personnel, les clients, le stock et les statistiques depuis l'application.

Ce projet a été réalisé selon un cahier des charges très précis qui nous a été communiqué.

Ce projet m'a permis de travailler de nombreuses compétences.

Technologies utilisées : Programmation orientée objet (C++) & Création/Interrogation d'une Base de données
